# My Personal Portfolio: AI & ML Student Showcase

## Project Description

This is a static personal portfolio website developed as part of the **Computer Science Fundamentals & Career Pathways** assignment at K R Mangalam University. Built with foundational **HTML** and **CSS**, this project serves as a live digital resume to showcase my skills as a B.Tech CSE student specializing in **Artificial Intelligence and Machine Learning**.
